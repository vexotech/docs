## How It Works

Vexo Analytics works by listening to React Native events such as:
- App state changes (foreground/background)
- Screen transitions
- User interactions (taps, gestures)
- Device orientation changes
- Network requests
- Error occurrences

All this data is collected, processed, and sent to our secure servers where it's transformed into actionable insights through our intuitive dashboard.
