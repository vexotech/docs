# Overview

## Out-of-the-box

These metrics are available instantly, without any additional configuration:

**1. Stats Dashboard**

- Active Users - daily, weekly, and monthly active users.
- Session Time - average time spent in the app.
- Downloads - track installs over time.
- Total Sessions - total number of sessions across a given timeframe
- OS / Device Distribution - see which platforms and operating systems your users are on.
- App Intensity - total amount of events that happen in your app and distributing them over the hours of the week.
- Version Adoption - measure how quickly users update to the latest version.
- Geographic Insights - discover where your users are located.
- Popular Screens - identify the screens with the most engagement.
- Customizable:
    - Funnels - build funnels from screens or events.
    - Track Events - visualize and filter your custom events.

**2. People**

- See users’ key info at a glance: last interaction, app version, device model, country, total sessions and replays.
- If you are using Vexo Web: Unified user profiles across React Native & Web.

**3. Session Replays & Heatmaps**

- Session Replays - watch real user sessions to understand their interactions.
- Heatmaps - visualize the most engaged areas of your app.

## Custom

For deeper, product-specific insights you can:

- Custom Events — track specific user actions that matter most.
- Custom Dashboards — create and personalize your dashboards for visualizing and filtering your custom events.
