# Web Integration Guide

Get Vexo Analytics running on your website in under 5 minutes.

## Installation Options

### Option 1: Script Tag (Recommended)

Add this script to your HTML `<head>` section:

```html
<script>
  (function(v,e,x,o,s,c,r,i,p,t){v.VexoObject=s;v[s]=v[s]||function(){
  (v[s].q=v[s].q||[]).push(arguments)};v[s].l=1*new Date();c=e.createElement(x),
  r=e.getElementsByTagName(x)[0];c.async=1;c.src=o;r.parentNode.insertBefore(c,r)
  })(window,document,'script','https://cdn.vexo.co/vexo.js','vexo');

  vexo('init', 'YOUR_API_KEY');
</script>
```

### Option 2: NPM Package

For modern web applications:

```bash
npm install vexo-web-analytics
# or
yarn add vexo-web-analytics
```

Then initialize in your app:

```javascript
import { initVexo } from 'vexo-web-analytics';

// Initialize with your API key
initVexo('YOUR_API_KEY');
```

## Framework-Specific Integration

### React Integration

```jsx
// App.js
import { useEffect } from 'react';
import { initVexo } from 'vexo-web-analytics';

function App() {
  useEffect(() => {
    initVexo('YOUR_API_KEY');
  }, []);

  return (
    <div className="App">
      {/* Your app content */}
    </div>
  );
}
```

### Vue Integration

```javascript
// main.js
import { createApp } from 'vue';
import { initVexo } from 'vexo-web-analytics';
import App from './App.vue';

const app = createApp(App);

// Initialize Vexo Analytics
initVexo('YOUR_API_KEY');

app.mount('#app');
```

### Angular Integration

```typescript
// app.component.ts
import { Component, OnInit } from '@angular/core';
import { initVexo } from 'vexo-web-analytics';

@Component({
  selector: 'app-root',
  templateUrl: './app.component.html'
})
export class AppComponent implements OnInit {
  ngOnInit() {
    initVexo('YOUR_API_KEY');
  }
}
```

## Configuration Options

### Basic Configuration

```javascript
vexo('init', 'YOUR_API_KEY', {
  // Optional: Custom configuration
  trackPageViews: true,
  trackClicks: true,
  enableSessionReplay: false,
  respectDoNotTrack: true
});
```

### Advanced Configuration

```javascript
vexo('init', 'YOUR_API_KEY', {
  // Privacy settings
  respectDoNotTrack: true,
  enableCookies: false,

  // Tracking settings
  trackPageViews: true,
  trackClicks: true,
  trackScrollDepth: true,

  // Performance settings
  batchSize: 10,
  flushInterval: 5000,

  // Debug mode
  debug: process.env.NODE_ENV === 'development'
});
```
